## Ghoost - Personal AI Manager

Ghoost is a simple command-line personal manager. It helps you keep track of tasks, notes, and reminders, all stored locally in a JSON file.

### Usage

```
python ghoost.py add-task "Buy groceries"          # add a task
python ghoost.py list-tasks                        # list all tasks
python ghoost.py done 1                            # mark task #1 as done
python ghoost.py add-note "Remember to call mom"   # save a note
python ghoost.py list-notes                        # list all notes
python ghoost.py add-reminder "Pay rent" 2026-07-01 # add a reminder
python ghoost.py list-reminders                    # list all reminders
```
